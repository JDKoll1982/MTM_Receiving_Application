# Tech Stack

**Core:**

- **Language:** C# 12
- **Platform:** .NET 8
- **UI Framework:** WinUI 3 (Windows App SDK 1.6+)

**Libraries:**

- **MVVM:** `CommunityToolkit.Mvvm` (v8.2.2)
- **UI Controls:** `CommunityToolkit.WinUI.UI.Controls` (v7.1.2)
- **Dependency Injection:** `Microsoft.Extensions.DependencyInjection` (v8.0.0)
- **Hosting:** `Microsoft.Extensions.Hosting` (v8.0.0)
- **Database:** `MySql.Data` (v9.4.0)

**Testing:**

- **Framework:** xUnit
- **Mocking:** Moq
- **Runner:** `xunit.runner.visualstudio`

**Database:**

- **Engine:** MySQL 8.0
- **Access:** ADO.NET with Stored Procedures
