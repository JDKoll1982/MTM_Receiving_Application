# Window Sizing - User Guide

## Overview

The MTM Receiving Application uses standardized window sizes for consistent display.

## Window Sizes

- **Main Window**: 1200×800 pixels (resizable)
- **Dialogs**: 400-600 pixels wide (non-resizable)
- **Splash Screen**: 500×450 pixels (non-resizable)

## Display Requirements

- **Recommended**: 1920×1080 (Full HD)
- **Minimum**: 1366×768 (HD)
- **DPI Scaling**: Supports 100%, 150%, 200%

## Troubleshooting

If windows appear too large or small:
1. Check Windows display scaling settings
2. Adjust to 100% or 150% scaling
3. Restart application
