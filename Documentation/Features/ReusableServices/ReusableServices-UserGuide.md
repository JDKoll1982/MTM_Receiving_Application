# Reusable Services - User Guide

## Overview

This document lists services that can be reused in other MTM projects.

## Core Services

1. **Logging Service** - CSV-based logging
2. **Error Handler** - Centralized error handling
3. **Database Helper** - MySQL/SQL Server stored procedure execution

These services handle common tasks and can be copied to new projects.

For setup instructions, see the DeveloperGuide.
