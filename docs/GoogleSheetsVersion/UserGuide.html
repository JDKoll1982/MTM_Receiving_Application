üöÄ How to Use the System (Step by Step)

üìù Daily Data Entry

  1. Choose your sheet: Click on either "Expo" or "Vits" tab at the bottom
  2. Start entering data in row 2:
    • Column A: Enter the quantity (e.g., 100, 250, 1500)
    • Column B: Enter the Material ID (e.g., FHS-0420-12, 25086399)
    • Column C: Enter PO Number if you have one
    • Column E: Enter Heat number
    • Column G: Enter where you put the item (e.g., RECV, STORE-A1)
  3. Don't worry about: Employee names, dates, or label numbers - these fill automatically
  4. Continue adding rows: Each new item gets its own row

üîÑ Clean Up Your Data

  1. Click the "Fill Blank Spaces" checkbox (Row 3)
  2. Watch the progress: J3 will show the progress
  3. Result: All missing employee names and dates are filled in

üìä Organize for Printing

  1. Click the "Sort for Printing" checkbox (Row 4)
  2. Watch the progress: J4 will show the sorting progress
  3. Result: Data is organized by Material ID and Heat frequency

üíæ Save to Permanent Storage

  1. Click the "Save to History" checkbox (Row 2)
  2. Watch the progress: J2 will show the transfer progress
  3. Result: All data moves to History 2025 sheet, current sheet is cleared for new entries
‚ö†Ô∏è Important: Once you save to history, the data is permanently moved. Make sure your data is complete and correct before saving!

üìà Understanding the Summary Section

Below your data (starting around row 10), you'll see a "Total Item Weights" section that automatically calculates:

  • Unique part counts: How many different Material IDs you have
  • Total quantities: Combined quantities for each Material ID
  • Real-time updates: Changes automatically as you add or modify data
Note: This summary automatically excludes header rows and only counts actual inventory data.

üéâ Quick Start Summary

For new users - your first 5 minutes:
  1. Click on "Expo" or "Vits" tab
  2. Start typing in row 2: Quantity, Material ID, PO Number, Heat, Location
  3. Add a few more rows of data
  4. Click "Fill Blank Spaces" checkbox and wait for completion
  5. Click "Sort for Printing" checkbox to organize
  6. When finished for the day, click "Save to History"

That's it! You're now managing inventory like a pro! üöÄ
