# Feature Removal Request Outputs

Last Updated: 2026-03-22

Save generated feature removal request exports in this folder.

Suggested naming:

- `YYYY-MM-DD-short-name.export.md`
- `YYYY-MM-DD-short-name.export.json`

When asking Copilot to remove or retire the feature safely, link the saved export and use `.github/prompts/copilotforms-feature-removal-request.prompt.md`.
