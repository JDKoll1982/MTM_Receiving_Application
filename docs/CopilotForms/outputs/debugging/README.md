# Debugging Outputs

Last Updated: 2026-03-18

Save generated debugging exports in this folder.

Suggested naming:

- `YYYY-MM-DD-short-name.export.md`
- `YYYY-MM-DD-short-name.export.json`

When asking Copilot to debug the issue, link the saved export and use `.github/prompts/copilotforms-debugging.prompt.md`.
