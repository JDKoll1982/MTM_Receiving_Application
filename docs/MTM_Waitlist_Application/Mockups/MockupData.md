Operator Request Types - Directed twards a Material Handler for fulfillment of the request. Each request type should be linked to a work order or part number where applicable.

Coils (Can be requested to be brought to Press or returned to Storage)
Flatstock (Can be requested to be brought to Press or returned to Storage)
Needs (COmpone Parts, Dunnage, Finished Goods) - linked to work order or part number
Returns (Component Parts, Dunnage, Finished Goods) - linked to work order or part number
Pickup Non Conforming Material (NCM) - linked to work order or part number, NCM Tasks should be kept separate from standard material handling tasks as they need a seprate process for tracking and resolution.
Empty Scrap Containers - linked to work order or part number (Scrap Metal Types: 5052 Aluminum, 3003 Aluminum, Steel, Stainless, Copper, Other)
Bring Dies to Press or Return to Storage - linked to work order or part number, can be multiple dies per request also give the material handler the die's location on the material hander dashboard.
Other - Freeform text input for special requests not covered by standard categories
