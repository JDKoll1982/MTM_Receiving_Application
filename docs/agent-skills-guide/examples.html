Examples

Practical skills you can add to this repo

How to Use This Page

Each example below includes:

  • A ready-to-copy SKILL.md frontmatter description (the most important part for discovery)
  • Suggested scripts/, references/, templates/, and optional assets/
  • Example prompts that should activate the skill
Note: These are examples for learning and planning. Create the actual skill folders in .github/skills/ when you're ready.

Example 1: mvvm-architecture-enforcer (Recommended)

Goal: Prevents the most common architecture violations (ViewModel ‚Üí DAO calls, runtime Binding, non-partial ViewModels).

SKILL.md frontmatter (copy/paste) 
---
name: mvvm-architecture-enforcer
description: Enforces MVVM architecture rules for WinUI 3 applications. Use when creating or modifying ViewModels, Services, DAOs, or XAML bindings. Prevents ViewModels from calling DAOs directly, requires partial ViewModels inheriting from ViewModel_Shared_Base, enforces Service layer usage, and converts {Binding} to {x:Bind}.
license: MIT
---

Suggested resources

  • scripts/validate-mvvm.ps1 - scan ViewModels for DAO usage, scan XAML for {Binding}
  • references/architecture-rules.md - concise rules + examples
  • templates/viewmodel-template.cs - ViewModel scaffolding
  • templates/service-template.cs - IService_*/Service_* scaffolding

Example prompts

  • "Create a new ViewModel for a settings page"
  • "Audit this ViewModel for architecture violations"
  • "Convert these XAML bindings to x:Bind"

Example 2: mysql-stored-proc-dao-generator

Goal: Generates DAO methods that call MySQL stored procedures and return Model_Dao_Result without throwing exceptions.

SKILL.md frontmatter (copy/paste) 
---
name: mysql-stored-proc-dao-generator
description: Generates instance-based MySQL DAO classes and methods using stored procedures only. Use when creating or modifying DAOs for MySQL. Produces Model_Dao_Result returns, never throws from DAOs, maps MySqlParameter arrays, and uses Helper_Database_StoredProcedure.
license: MIT
---

Suggested resources

  • references/dao-rules.md - "never throw", "stored proc only", "Model_Dao_Result"
  • templates/dao-template.cs - constructor + error handling scaffold
  • templates/sp-call-template.cs - standard MySqlParameter mapping pattern
  • scripts/find-raw-sql.ps1 - searches for SQL strings in C# files

Example prompts

  • "Create a DAO method for sp_Receiving_Line_Insert"
  • "Convert this raw SQL to stored procedure usage"

Example 3: xaml-xbind-auditor

Goal: Detects runtime bindings and guides conversion to compile-time x:Bind with explicit Mode.

SKILL.md frontmatter (copy/paste) 
---
name: xaml-xbind-auditor
description: Audits WinUI 3 XAML for binding issues. Use when reviewing or editing XAML. Detects runtime {Binding}, missing x:Bind Mode, missing x:DataType in DataTemplates, and suggests fixes to enforce compile-time binding.
license: MIT
---

Suggested resources

  • scripts/find-runtime-binding.ps1 - find {Binding} usage
  • references/xbind-patterns.md - examples for TextBox Mode=TwoWay and UpdateSourceTrigger

Example prompts

  • "Audit this XAML file for binding issues"
  • "Convert these {Binding} to {x:Bind} and set Mode"
