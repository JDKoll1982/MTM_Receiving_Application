# Agent Skills Guide (HTML)

Last Updated: 2026-01-24

This folder contains a small, static HTML documentation site that explains GitHub Copilot Agent Skills for the MTM Receiving Application repo.

## Pages

- `index.html` - Landing page
- `what-are-skills.html` - Plain-English overview of skills
- `getting-started.html` - Step-by-step tutorial (uses `mvvm-architecture-enforcer` example and demonstrates `scripts/`, `references/`, `templates/`)
- `folder-structure.html` - Explains skill locations and folder purposes (`scripts/`, `references/`, `assets/`, `templates/`)
- `examples.html` - Several skill ideas with copy/paste frontmatter
- `best-practices.html` - How to write descriptions, structure skills, bundle resources, and troubleshoot

## Shared assets

- `styles.css` - Shared styling (light/dark theme, responsive layout, accessibility)
- `app.js` - Shared behavior (theme toggle, active nav highlight, smooth scrolling, code-copy helper)

## How to view locally

Open `index.html` in a browser.

If you prefer running a local server, any static file server works (PowerShell, Python, etc.).
