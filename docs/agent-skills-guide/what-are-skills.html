What Are Agent Skills?

A plain-English explanation for beginners

The Simple Answer

Agent Skills are folders of instructions that teach GitHub Copilot how to do specialized tasks. Think of them as "lesson plans" that make Copilot smarter at specific jobs.

Real-world analogy: If Copilot is like a smart assistant, Skills are like training manuals that teach it new capabilities ‚Äî similar to how you might give a new employee a handbook for completing specific tasks.

Why Do They Exist?

Out of the box, GitHub Copilot knows a lot about programming. But your team probably has:

  • Specific workflows unique to your project
  • Company branding guidelines
  • Custom testing procedures
  • Specialized tools and scripts
  • Domain-specific knowledge

Agent Skills let you teach Copilot about these things once, and then Copilot applies that knowledge automatically whenever it's relevant.

How Are Skills Different from Instructions?

Feature Custom Instructions (.instructions.md) Agent Skills (SKILL.md)
Purpose Define coding standards and style rules Enable task-specific workflows
When Loaded Always active for matching files Only when relevant to the task
Can Include Text instructions only Instructions + scripts + templates + examples
Best For "Always use async/await" type rules "How to test web apps" type workflows
Example C# coding conventions for all .cs files Playwright testing toolkit with scripts

Who Can Create Skills?

Anyone! You don't need to be an AI expert. If you can:

  • Write clear instructions in plain English
  • Create a folder and a text file
  • (Optional) Write simple scripts

...then you can create Agent Skills.
